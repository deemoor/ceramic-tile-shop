# Test Assignment: The Artisan Kiln

An e-commerce application for ordering ceramic tiles with an interactive design tool.

**Demo:** https://ceramic-tile-shop.vercel.app/

### Local Launch

1. ```npm install```
2. ```npm run dev```

### Tech Stack
* Framework: Next.js 16, React 19.
* Language: TypeScript
* State Management: Redux Toolkit
* Styling: Tailwind CSS
* Validation: Zod
* Testing: Vitest
* Drag & Drop: dnd-kit
